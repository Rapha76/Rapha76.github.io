Optimisation de la sélection des souches pour la création d'un ferment


Présentation du projet

Danone vise à réduire le sucre ajouté dans ses yaourts tout en préservant leurs caractéristiques. Des recherches sont menées pour trouver des solutions, comme l'utilisation de bactéries lactiques moins post-acidifiantes, afin de maintenir l'équilibre entre sucre et acidité. Cette étude vise à collecter des données en vue de les analyser et de construire une base de données facilitant la prise de décision pour choisir les souches les plus adaptées à la création d'un nouveau ferment.


Organisation du travail - Démarche de solution

Le processus de création de nouveaux ferments est long et complexe, nécessitant des efforts empiriques pour modéliser les interactions entre les souches. Il est donc possible d’optimiser ce processus en exploitant les données.

1ère étape : Collecte des données. Les données sur les souches sont rassemblées à partir de divers fichiers Excel non standardisés. Des efforts sont déployés pour normaliser et consolider ces données, notamment en utilisant Python pour leur manipulation.

2ème étape : Nettoyage des données. Des analyses statistiques sont effectuées pour évaluer la qualité des données, identifier les valeurs aberrantes et standardiser les formats. Les données sont nettoyées et harmonisées pour faciliter leur comparaison et leur analyse ultérieure.

3ème étape : Création d'une Template. Une Template est développée pour standardiser la structure des données sur les souches et les ferments, simplifiant ainsi la collecte et l'analyse des données pour les futurs essais.

4ème étape : Analyse des données. Des visualisations, telles que des nuages de corrélation, sont utilisées pour explorer les relations entre les variables des différentes bases de données.

5ème étape : Conception d'un tableau de bord. Un tableau de bord est créé pour visualiser les données de manière claire et faciliter la prise de décision.


Conclusion

Le projet a surmonté divers défis liés à la gestion et à l'analyse des données, grâce à l'utilisation de Python et à l'adoption d'une approche systématique. La création d'une Template standardisée représente une avancée importante pour les futurs projets. Les prochaines étapes incluront l'utilisation de modélisations telles que l'ACP et des techniques d'optimisation pour affiner les résultats.
