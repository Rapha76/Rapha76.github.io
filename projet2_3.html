Collecte automatisée de données web


Présentation du projet

L’objectif de cette SAE était de mettre en place une API à l’aide de la librairie Django à partir de données récupérées sur une page Wikipédia. Ce projet s’est déroulé en 3 grandes étapes : scraping de données, chargement des données dans une base de données et la mise en place de l’API.


Organisation du travail - Démarche de solution

Ce projet devait être réalisé de manière individuelle. Avant d'entamer le processus d'extraction de données, j'ai effectué des recherches pour comprendre les étapes nécessaires, en particulier sur l'utilisation de la bibliothèque BeautifulSoup. Etant donné que les informations les plus cruciales étaient présentées dans un tableau, j’ai extrait ces données pour les organiser dans un DataFrame. Cette approche a considérablement simplifié la phase d’insertion des données dans ma base de données MySQL. Par la suite, j’ai déployé une API Django pour faciliter l’accès et la manipulation de ces données.


Apports du projets et objectifs

Ce projet m'a offert une compréhension approfondie de l'extraction de données à partir d'une page Web et de la mise en place d'une API. Cependant, en raison de difficultés significatives dans la configuration de l'API, je suis conscient de la nécessité de persévérer dans mes efforts afin de surmonter ces obstacles et d'assurer une mise en œuvre sans difficultés à l'avenir.
